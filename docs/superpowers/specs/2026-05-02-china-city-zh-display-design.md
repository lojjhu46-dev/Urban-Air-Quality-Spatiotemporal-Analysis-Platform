# 中国省市中文显示设计

**目标**

在切换到 `zh-CN` 时，历史数据 Agent 页面中的中国省份和城市显示为中文，同时继续保留英文查询值用于地理解析和 API 调用。

**约束**

- 只调整中国条目的显示层。
- 不改变 `AgentCityOption` 现有的英文查询语义。
- 不扩展到其他国家/地区的地名翻译。

**方案**

1. 在中国城市目录中补充省份和城市的中文显示映射。
2. 在 `AgentCityOption` 中增加语言感知的显示方法，用于省份、城市和路径文本。
3. 页面下拉框继续使用英文稳定值作为 `options`，仅在 `format_func` 中按语言显示中文。
4. 为中国条目补回归测试，验证中文显示和英文查询值可同时成立。
